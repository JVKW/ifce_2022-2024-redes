Respostas da Atividade

1. Quais cores você utilizou nos diversos elementos do formulário (códigos em hexadecimal)?

  • Cor de fundo do contêiner: #f6fdfe
  • Cor do texto dos rótulos e título: #333 (título), #555 (rótulos)
  • Cor da borda dos campos de entrada: #ccc
  • Cor de fundo do botão: #1a8819
  • Cor do texto do botão: #fff
  • Cor de fundo do botão ao passar o cursor: #156e14

2. O que você teve que aplicar no CSS para que os elementos fossem apresentados um abaixo do outro?

Para garantir que os elementos fossem apresentados um abaixo do outro, apliquei a propriedade display: flex; junto com flex-direction: column; ao elemento form. Isso faz com que todos os elementos filhos do formulário se alinhem verticalmente.

3. Como você estilizou os elementos input?

Os elementos input foram estilizados da seguinte forma:

  • Largura total: width: 100%;
  • Preenchimento interno: padding: 10px 0;
  • Margem inferior: margin-bottom: 5px;
  • Borda: border: 1px solid #ccc;

4. Como você estilizou o botão de submit?

O botão de submit foi estilizado da seguinte forma:

  • Preenchimento: padding: 12px 20px;
  • Borda: border: none;
  • Cantos arredondados: border-radius: 14px;
  • Cor de fundo: background-color: #1a8819;
  • Cor do texto: color: #fff;
  • Cursor: cursor: pointer;
  • Transição de cor: transition: background-color 0.3s ease;
  • Cor de fundo ao passar o cursor: background-color: #156e14;

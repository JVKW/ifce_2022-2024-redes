Respostas da Atividade

1. Quais as principais teorias utilizadas para definição de cores primárias?

  • Teoria aditiva (RGB): As cores primárias são vermelho (Red), verde (Green) e azul (Blue). Combinadas em diferentes intensidades, produzem outras cores, e misturadas completamente, produzem o branco.
  • Teoria subtrativa (CMY): As cores primárias são ciano (Cyan), magenta (Magenta) e amarelo (Yellow). Quando combinadas, absorvem luz, e ao misturar completamente, produzem o preto.

2. Qual a diferença entre sistema subtrativo e sistema aditivo?

O sistema aditivo (RGB) combina luz para criar cores, sendo usado em telas. Começa com preto e, ao adicionar luz (vermelha, verde, azul), cria cores. O sistema subtrativo (CMY) começa com branco e, ao adicionar pigmentos ou tintas, absorve luz para criar cores, sendo usado em impressão.

3. Qual a diferença entre sistema RGB e sistema CMY?

O sistema RGB é aditivo, utilizado em dispositivos eletrônicos, onde as cores são criadas pela combinação de luz. O sistema CMY é subtrativo, usado em impressão, onde as cores são criadas pela absorção de luz.

4. Quais os principais aspectos de acessibilidade a serem levados em consideração na definição da paleta de cores de um site ou aplicação?

Os principais aspectos incluem contraste adequado entre o texto e o fundo, evitar o uso exclusivo de cores para transmitir informações, e garantir que as cores sejam distinguíveis por pessoas com daltonismo.

5. Explique de forma breve quais as Diretrizes de Acessibilidade para Conteúdo Web (WCAG) aplicadas à definição de cores.

As WCAG recomendam um contraste mínimo de 4.5:1 para textos normais, o uso de cores em conjunto com outros elementos visuais e a verificação da acessibilidade para pessoas com deficiências visuais.

6. O que é daltonismo (discromatopsia) e quais os tipos de daltonismo? Como podemos trabalhar a acessibilidade na Web relacionada às pessoas que possuem essa condição?

O daltonismo é a incapacidade de distinguir certas cores. Existem três tipos principais:

  • Protanopia: dificuldade em ver tons de vermelho.
  • Deuteranopia: dificuldade em ver tons de verde.
  • Tritanopia: dificuldade em ver tons de azul.

Na web, podemos melhorar a acessibilidade usando paletas de cores amigáveis, aumentando o contraste e adicionando padrões ou símbolos além das cores.
